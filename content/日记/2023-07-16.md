1. 配置邮箱和用户名
----

在 Git Bash 中执行以下指令，将邮箱和用户名配置为正确的值：


```bash
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
```
请将 `you@example.com` 替换为你的邮箱地址，将 `Your Name` 替换为你的名字。

2. 生成密钥对
--------

在 Git Bash 中运行以下命令，生成 RSA 密钥对：

一路按回车键

```bash
ssh-keygen -t rsa
```


3. 添加公钥到 GitHub
------------

打开 GitHub 地址(https://github.com/settings/ssh/new/)

添加公钥时，GitHub 将会显示一个表单，你可以将公钥添加到表单中。


```
~/.ssh/id_rsa.pub
```
然后按下 "Add key" 按钮。

4. 验证是否关联成功
------------

在 Git Bash 中执行以下命令，验证是否关联成功：


```bash
ssh -T git@github.com
```
执行此命令后，如果一切正常，将会看到一个提示，表示 SSH 连接成功。
