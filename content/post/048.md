安装下载工具:

```shell
apt update -y  && apt install -y curl
```

  

一键脚本工具:

```shell
curl -sS -O https://raw.githubusercontent.com/kejilion/sh/main/kejilion.sh && chmod +x kejilion.sh && ./kejilion.sh
```

根目录执行

```bash
nano .bashrc
```


在内容最下方添加这行数据，定义k按键调用`kejilion.sh`

```shell
alias k='curl -sS -O https://kejilion.pro/kejilion.sh && chmod +x kejilion.sh && ./kejilion.sh'
```

![Pasted image 20230815141922](https://r2.leshans.eu.org/2023/08/4f4f7aea12ec40ecab2446aad75434fb.webp)

退出nano编辑器, `ctrl+x` ` y`  回车

更新下文件

```
source ~/.bashrc
```
