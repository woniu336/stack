为什么是 7.7.0 版本，而不是其他版本呢？因为这个版本没有很旧，也没有太多限制，还可以破解

> ### **安装宝塔：**

```
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec
```

### **1、下载离线升级包**

```
wget https://github.com/woniu336/2btc/raw/main/LinuxPanel-7.7.0.zip
```

### **2、解压**

yum -y install unzip zip （若解压失败）

```
unzip LinuxPanel-7.7.0.zip
```

  

### **3、进入升级目录**

  

```
cd /root/panel
```

  

### **4、运行降级**

  

bash update.sh

  

### **5、去除登录**

  

```
rm -f /www/server/panel/data/bind.pl
```

  

全部安装完成后，执行以下内容，避免官方搞小动作

```
echo '127.0.0.1 bt.cn' >>/etc/hosts
```



> ### 其他方法

**Centos/Ubuntu/Debian 安装命令独立运行环境（py 3.7）**

```shell
curl -sSO https://raw.githubusercontent.com/woniu336/btpanel-v7.7.0/main/install/install_panel.sh && bash install_panel.sh
```

## 净化与破解

###  **修改 hosts 文件**

```shell
echo '127.0.0.1 bt.cn' >>/etc/hosts
```



### 解除强制手机登录

```shell
rm -f /www/server/panel/data/bind.pl
```

### **手动解锁宝塔所有付费插件为永不过期**

文件路径：`/www/server/panel/data/plugin.json`

搜索字符串：`"endtime": -1` 全部替换为 `"endtime": 999999999999`



###   给plugin.json文件上锁防止自动修复为免费版

```shell
chattr +i /www/server/panel/data/plugin.json
```



注意事项 : 服务器重启后又回到"endtime": -1 需要再次修改替换
